Install, update and various other documentation can be found in the "docs"
folder/directory.

.:IMPORTANT:.
Some versions of makale require database structure changes when being
updated. These changes are made via an easy to use script included in 
this archive.

Please see the text file docs/UPDATE.txt for more information on updating
makale and if you need to run an update script. If you are install 
makale for the first time, or re-installing afresh, you need not worry 
about this.
